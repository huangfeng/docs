removeSpan
=============

@short: removes a rowspan/colspan from the datatable

@edition:pro

@params:
- id		id		row id that starts a colspan/rowspan
- column	string				column id that starts a colspan/rowspan

@example:
grid.removeSpan(1, "country");
grid.refresh();
