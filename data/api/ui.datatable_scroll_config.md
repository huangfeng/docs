scroll
=============


@short:
	enables or disables scroll for the datatable

@type: boolean,string
@example:

webix.ui({
    view:"datatable",
    scroll:false
});

@values:
- y (true)	 enables vertical scroll (the default value) 
- x	enables horizontal scroll
- xy	enables both scrolls
- false	 disables both scrolls



@template:	api_config
@descr:


Enabled by default
