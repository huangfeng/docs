autoheight
=============


@short: lets the component autosize to its contents
	

@type:  boolean
@example:

webix.ui({
rows:[
        { template:"html->my_box1", autoheight:true},
        { template:"Area 1"},
        { template:"html->my_box2", autoheight:true}
      ]
})
