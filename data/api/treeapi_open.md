open
====

@short: opens the branch with the specified id
	

@params:
- id	id	the node id
* show	boolean		if true, opens all parent of the branch


@example:
tree = new webix.ui({
	view: "tree",
	data: [
		{ id: "branch1", value: "The Shawshank Redemption", data: [
			{ id: "1.1", value: "Part 1" },
			{ id: "1.2", value: "Part 2" }
		]},
		{ id: "branch2", value: "The Godfather", data: [
			{ id: "2.1", value: "Part 1" }
		]}
	]
});	

tree.open("branch1");
