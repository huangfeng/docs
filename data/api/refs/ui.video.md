ui.video 
=============


{{memo Video player view. }}

### Constructor

~~~js
	var video = webix.ui({
		view:"video", 
		src:"film.mp4"
	});
	//or, in case of jQuery
	$("#mydiv").webix_video({
		...config options goes here..
	});
~~~
